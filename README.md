# DonutJL
Julia implementation of this wonderful [article](https://www.a1k0n.net/2011/07/20/donut-math.html). Generates a video instead of ASCII characters.

## Usage:

```
julia donut.jl
```

## System dependencies:
    1. FFMPEG
## Julia dependencies:
    1. ArgParse
    2. Images
    3. ImageFiltering
    4. ImageTransformations
    5. ProgressMeter
