# DonutJL
Julia implementation of this wonderful [article](https://www.a1k0n.net/2011/07/20/donut-math.html). 

## Usage:

```
julia -p <NUM_PROCESSES> donut.jl --mode=graphics
julia -p 1 donut.jl --mode=terminal
```

## System dependencies:
    1. FFMPEG
## Julia dependencies:
    1. ArgParse
    2. Images
    3. ImageFiltering
    4. ImageTransformations
    5. ProgressMeter
